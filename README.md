# kiodo-pks-test
PKS Tests and demo kit



## Networking

To demo networking

```
kubectl create namespace kiodo
```

To see the deployment

```
kubectl get deployments --namespace=kiodo
```


```
kubectl get networkpolicies --namespace=kiodo
```
